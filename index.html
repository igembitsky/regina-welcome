03
Install Developer Tools

Two quick things to install behind the scenes. You'll paste some commands into Terminal — that black window from earlier. Just copy, paste, and hit Enter.

  1. A

    Open Terminal

    Press ⌘ Space (hold the Command key and tap Space), type "Terminal", and open the app that appears. Terminal is a text-only way to talk to your Mac — you type commands and it does things. It looks like a black or white window with text.

  2. B

    Install Homebrew

    Homebrew is a helper that installs other tools for you. Copy the line below, paste it into the Terminal window (⌘V), and press Enter:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    Terminal — zsh
    ~ % /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ==> Downloading and installing Homebrew...
    ==> Installation successful!
     
    ✓ Homebrew installed

    It will ask for your Mac password (the one you use to log in). When you type it, nothing will appear on screen — no dots, no stars, nothing. That's normal! Just type your password and press Enter.

  3. C

    Install Git

    Git keeps a history of every change you make, so you can always go back if something breaks. Install it by pasting this into Terminal:

    brew install git
    Terminal — zsh
    ~ % brew install git
    ==> Downloading git...
    ==> Pouring git--2.47.0.arm64_sequoia.bottle.tar.gz
    ✓ git installed successfully
  4. D

    Set your name in Git

    This just tells Git your name and email so your work is labeled as yours. Paste these two lines into Terminal (replace the email with yours):

    git config --global user.name "Regina Shi" git config --global user.email "your-email@example.com"

    Use the same email you used when signing up for GitHub.

05
Install Superpowers

This gives Claude extra abilities — like showing you visual design options in your browser, planning your project step-by-step, and more.

  1. A

    Open the Claude Code panel in VS Code

    Look for a chat-like area at the bottom or side of VS Code — it should have appeared after you installed Claude Code in the previous step. Click on it to open it.

  2. B

    Type the install command

    In the Claude Code chat area, type this into the text box at the bottom and press Enter:

    claude install superpowers
