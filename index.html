Install Developer Tools

Open your Mac's Terminal and install Homebrew + Git. Follow each sub-step.

  1. A

    Open Spotlight Search

    Press ⌘ + Space on your keyboard. This opens the Mac Spotlight search bar.

  2. B

    Type "Terminal" and hit Enter

    This is what you'll see:

    Terminal
    🖥️
    Terminal
    Application — /System/Applications/Utilities
  3. C

    Install Homebrew

    Paste this command into Terminal and hit Enter. It installs Homebrew, a Mac package manager.

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    ⏱ This may take a few minutes. Follow the prompts if asked for your password.

  4. D

    Install Git & verify

    Once Homebrew is done, run this to install Git and confirm it works:

    brew install git && git --version

Here's what a successful install looks like:

Terminal — zsh
~ $ brew install git && git --version
==> Downloading https://ghcr.io/homebrew/core/git
✓ git 2.44.0 installed successfully
git version 2.44.0

Install Superpowers

This plugin gives Claude advanced skills — brainstorming, planning, TDD, and more. It's how you'll design your app before writing a line of code.

  1. A

    Open the Claude Code panel in VS Code

    You should see it in the bottom panel or sidebar after installing the extension.

  2. B

    Type the install command

    In the Claude Code chat input at the bottom, type this and hit Enter:

    claude install superpowers

    Here's where you'll type it:

    Claude Code — VS Code
    Claude Code
    Terminal
    Output
    C

    Welcome, Regina! I'm Claude, your AI coding assistant.

    claude install superpowers
    Type in the chat input at the bottom, then hit Enter
  3. C

    Verify skills are loaded

    Open Terminal and run:

    claude skills

    You should see skills like brainstorming, writing-plans, frontend-design, and more.
